# CRESidekick

AI-native CRM for commercial real estate brokers, teams, and brokerages.

## Vision

CRESidekick is an AI-first CRM that replaces fragmented, overpriced CRE tools with a unified platform at 1/3 the cost. Built from the ground up with AI at the core — not bolted on.

## Key Differentiators

- **AI-Native** — Conversational CRM interface, AI-generated documents, automated data entry
- **Integrated BI** — Agent-level business intelligence dashboards (GCI, pipeline velocity, lead source ROI)
- **Affordable** — $29–49/user/month vs $79–249 for incumbents
- **Mobile-First** — Native app with offline capability
- **Cross-Vertical** — Handles both residential and commercial real estate
